# Z-BlogPHP

`Z-BlogPHP` 基于 `PHP` 平台，支持 `MySQL`(含 MariaDB)、`SQLite` 和 `PostgreSQL` 数据库。期待它能成为您建站、写博的上佳选择。

> 我们一直在努力把 Z-BlogPHP 打造成高可玩性、乐高积木式的网站程序！
